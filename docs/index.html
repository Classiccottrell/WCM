The 30-Second Version

What this is, in plain English

★ In simple terms

Every property shoot produces 200–400 photos. Until now, one person — Trent — had to look through every single one to pick the "hero" images that lead the marketing. That's the slowest step in getting a listing live.

This tool does the first pass automatically. It looks at all the photos, scores each one against West Coast Modern's signature style, and produces a tidy report with its top picks. Trent now reviews 3–5 options instead of 300. The final choice is always still a human's — the tool just removes the heavy lifting.

It runs through Claude (an AI assistant). You can use it the easy way — in a web browser with no setup — or the automatic way, where it runs on a computer and processes new shoots on its own.

What It Produces

How it works & what you get back

Three simple stages — triage, recommend, deliver — ending in a polished report a non-technical person can read at a glance.

★ In simple terms

Think of it like a talented assistant doing a first edit. It quickly rates every photo, pulls the strongest contenders forward, and writes up a neat summary telling you which image should lead, which are backups, and which to skip — and why.

Stage 01

Fast triage

Every image is quickly scored for how well it fits the West Coast Modern look. Nothing gets missed, and weak shots are filtered out automatically.

Stage 02

Hero recommendations

The strongest candidates are studied more closely and ranked. The very best are surfaced as the lead "hero" images for the campaign.

Stage 03

Ready-to-use report

You receive a branded report — top picks, a shortlist, and clear guidance — without anyone manually sifting through hundreds of photos.

Every report includes

The selects

  • Hero (horizontal) — the lead image for website, email header and MLS
  • Hero vertical — for Instagram stories and email grids
  • Hero 2 — a backup / second-campaign option
  • Shortlist of 5–8 — for the gallery and tour sequence

The guidance

  • MLS set — up to 40 images, ordered and ready
  • Cuts list — what not to use, and the reason
  • Flags for Trent — close calls that genuinely need a human eye
  • Score sheet — every photo with its rating, in a simple spreadsheet

Getting Started

Setup instructions

Click any option to expand the full step-by-step. The technical steps each come with a plain-English note so anyone can follow along.

Option 1 — Browser (claude.ai) · 0 min setup
★ In simple terms

The no-setup option. Open a website, paste in the agent's instructions, drag in your photos, and ask it to run. Best for getting a result in the next five minutes.

Step 1 — Open Claude

Go to claude.ai (the Pro plan is recommended for larger batches of photos).

Step 2 — Load the agent's instructions

Paste the full contents of SKILL.md into a new conversation, like this:

I'm going to run a hero image selection session for West Coast Modern.
Here are your instructions:

[PASTE FULL CONTENTS OF SKILL.md HERE]

Ready to receive photos.

Step 3 — (Optional) Add Trent's taste reference

If Trent has filled in TASTE_REFERENCE_TEMPLATE.md, paste it next for results that more closely match his eye.

Step 4 — Upload photos and run

Property: [House Name]
Hook: [One sentence about what makes this property unique]
Target: [MLS | Instagram | Email | Feature Story | All]

[DRAG AND DROP PHOTOS — up to 20 at a time]

Run the hero selection.

Step 5 — Review and save

Claude returns the Hero Report. Review it, confirm the pick, and save it to the property folder. Tip: upload 20–30 photos per batch; for 200+, pull the top 40–50 exteriors first.

Option 2 — Desktop (Claude Cowork) · ~5 min setup
★ In simple terms

Same idea as Option 1, but using the Claude desktop app, which can read photos straight from a folder on your Mac. Nothing to upload — you just point it at the folder. Ideal for big shoots that already live on the computer.

Step 1 — Open Claude Cowork

Open the Cowork desktop app and sign in with your claude.ai account (Pro plan).

Step 2 — Load the instructions

Paste SKILL.md the same way as Option 1.

Step 3 — Point Claude at the folder

Instead of uploading, give it the folder location:

Property: [House Name]
Hook: [One sentence about what makes this property unique]
Target: [MLS | Instagram | Email | Feature Story | All]

Photos are in this folder on my Mac:
/Users/[yourname]/Dropbox/WCM/[PropertyName]/photography/raw/

Please read all images in that folder and run the hero selection.

Step 4 — Review together

Claude reads the images directly and produces the report. You can ask follow-up questions, request score breakdowns, or compare two specific photos before confirming.

Option 3 — Automatic (the wcm pipeline) · ~15 min setup
★ In simple terms

The hands-free option. After a one-time setup, a single command processes a whole folder of photos and writes the report right back into it — no browser, no uploading. It can even run on its own and react the moment a new shoot lands. This is the "set it and forget it" mode for recurring work, and it's the only option that needs a developer-style setup once.

What you need first

  • Python 3.9+ — free software the tool runs on (check with python3 --version)
  • An Anthropic API key — a private password that lets the tool talk to the AI, from console.anthropic.com
  • This project downloaded to the computer

One-time setup

In Terminal, from inside the Hero-Image-Agent folder:

# Create an isolated workspace and install the tool
cd Hero-Image-Agent
python3 -m venv .venv
source .venv/bin/activate
pip install -e .

# Give it your API key (the AI password)
export ANTHROPIC_API_KEY="sk-ant-..."

This installs a single command called wcm.

Check everything is ready

wcm doctor

Confirms Python, the dependencies, your API key (with a live test) and the skill file are all in place — and tells you exactly how to fix anything that isn't. Run this before your first real run.

Run it on a folder

# Standard run
wcm run /path/to/PropertyName

# Review more candidates, run faster
wcm run /path/to/PropertyName --top 20 --workers 10

# Stage 1 finished but Stage 2 failed? Pick up where you left off
wcm run /path/to/PropertyName --resume

What it writes back into the folder

  • triage_scores.csv — every image with its score, in a spreadsheet
  • hero_report.md — the full written Hero Report
  • hero_report.html — a styled, branded version you open in any browser

Hands-free watching (advanced)

Point it at a parent folder and it processes each new shoot automatically as it arrives — folders that already have a report are skipped.

wcm watch /path/to/Parent --scan-existing

It can also be set to start automatically when the Mac logs in, so it's always running in the background. Full details are in INSTALL.md and PIPELINE_README.md.

If something looks off

★ In simple terms

Most issues are quick fixes. The most common one for the automatic option is simply re-activating the workspace in a new Terminal window. And if the AI's taste ever feels off, the cure is more examples from Trent — not a code change.

SymptomFix
command not found: wcmRun source .venv/bin/activate first
ANTHROPIC_API_KEY is not setRun export ANTHROPIC_API_KEY="sk-ant-..."
No images foundCheck the folder path — images must be directly inside it, not in subfolders
Stage 2 timed outRe-run with --resume to retry just Stage 2
Scores feel wrongAdd more examples to TASTE_REFERENCE_TEMPLATE.md and re-run
Under The Hood

The technology, explained

For the curious — a non-technical tour of the moving parts.

★ In simple terms

The tool uses two AI "brains" to balance speed and quality: a fast, inexpensive one to skim every photo, and a stronger, more careful one to make the final hero call on the shortlist. Everything stays on your own computer and in your own cloud folders — there's no separate website or database to manage.

Two-stage AI review

A fast model triages the whole batch; a premium model studies the finalists. You get the speed of one and the judgment of the other — without paying premium cost on every photo.

Runs where the photos live

Reports are written straight into the property folder (which can be a synced Dropbox or Drive folder). No new system to log into, nothing extra to back up.
