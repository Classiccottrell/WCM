# West Coast Modern — AI Workflow Skills

Welcome to the **West Coast Modern AI Workflow Skills** repository. This project provides modular AI skill files (`skill.md`) and agent configurations (`agent.md`) designed to automate and orchestrate the creative production pipeline for architectural photography.

By leveraging a UX Skill Orchestra logic layer, this toolset is built to seamlessly move a property from raw photography to polished, published assets across Instagram, the website, email campaigns, and MLS. The goal is to dramatically increase delivery speed while maintaining consistent, high-quality aesthetic standards.

## 🎯 What This Does

This repository houses discrete AI agents and skills to handle distinct steps in the curation and publishing pipeline:

* **Hero Selection:** AI-assisted curation to identify the strongest hero exterior and interior shots from a comprehensive raw shoot.
* **Photo Sequencing:** Intelligently orders the selected images against the established Tour Sequence and MLS priority schema to tell a cohesive visual story.
* **Content Creation:** Generates platform-specific copy and image sets optimized for Instagram, Mailchimp, and the web.
* **Asset Management:** Automatically organizes and packages deliverables by property and platform for seamless handoff.
